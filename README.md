# Simulation of CO2 methanation in Python

Python scrip to simulate CO2 methanation along the length of a reactor, using pseudohomogeneous model. The PDE model is volume finite with the package FiPy.

It uses 4 different kinetics in order to confirm whether it is working correctly. It is not!

The code can be divided into eight sections which are as follows: “Imports and Numerical Utility functions”; “Species Definitions and Physical Properties”; “Kinetic Models”; “Kinetic Parameters”; “Conservation and Closure Functions”; “Reactor Model and Numerical Solver”; “Parametric Study Definition”; “Post-Processing and Visualization”. The RAW code is the whole code into one massive script.
