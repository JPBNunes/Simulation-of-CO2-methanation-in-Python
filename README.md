# Simulation of CO2 methanation in Python

Python scrip to simulate co2 methanation along the length of a reactor, using PH3 model. The PDE model is volume finite with the package FiPy.

It uses 4 different kinetics in order to confirm whether it is working correctly. It is not!

The code can be divided into eight sections which are as follows: “Imports and Numerical Utility functions”; “Species Definitions and Physical Properties”; “Kinetic Models”; “Kinetic and Physical Parameters”; “Conservation and Closure Functions”; “Reactor Model and Numerical Solver”; “Parametric Study Definition”; “Post-Processing and Visualization”. However, for the time being, it is only presented the code in its RAW form, without this subdivisions.
